# Legislacao: FGTS

## Normas vigentes

| Norma | Assunto | Impacto no sistema |
|---|---|---|
| Lei 8.036/90 | Lei do FGTS | Base geral: aliquota, deposito, saque |
| LC 150/2015 | FGTS Domestico | Obrigatoriedade para domesticos |
| Dec. 99.684/90 | Regulamento FGTS | Detalhamento operacional |
| Lei 13.932/2019 | FGTS Digital | Novo sistema de recolhimento |

## Aliquotas

| Categoria | Aliquota | Observacao |
|---|---|---|
| CLT geral | 8% | Sobre remuneracao bruta |
| Aprendiz | 2% | Lei 10.097/2000 |
| Domestico | 8% + 3,2% | 3,2% = antecipacao multa rescisoria |

## Incidencias

| Evento | Incide FGTS? | Observacao |
|---|---|---|
| Salario mensal | Sim | Base principal |
| Horas extras | Sim | Integra base |
| Ferias gozadas + 1/3 | Sim | Na competencia do gozo |
| Ferias indenizadas | Nao | Natureza indenizatoria |
| 13o salario | Sim | Cada parcela |
| Aviso previo indenizado | Sim | Deposito na rescisao |
| PLR | Nao | Isento |
| Abono pecuniario ferias | Nao | Isento |

## FGTS na rescisao

| Motivo | Multa | Saque |
|---|---|---|
| Sem justa causa | 40% | 100% |
| Acordo (art. 484-A) | 20% | 80% |
| Pedido demissao | 0% | Nao |
| Justa causa | 0% | Nao |
| Falecimento | 0% | 100% (herdeiros) |
| Aposentadoria | 0% | 100% |

## FGTS Digital

Desde 2024, o recolhimento e via FGTS Digital (substituiu SEFIP/GFIP).
O sistema gera arquivo para envio via w_guia_fgts_digital.srw.
