# Tasks

Pasta gerenciada automaticamente pelo parceiro de analise.

Cada arquivo JSON representa uma demanda em andamento ou concluida.
O sistema cria, atualiza e conclui tasks sem intervencao do analista.

Se voce fechar o Cursor e voltar, o sistema detecta tasks em andamento
e oferece retomar de onde parou.

**Nao edite estes arquivos manualmente.**
