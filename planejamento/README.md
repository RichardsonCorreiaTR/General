# Planejamento de Atualizacoes

Esta pasta contem os documentos de PLANEJAMENTO de cada versao do Projeto Filho:
specs, testes, revisoes, SWOT, diagnosticos.

Estes documentos sao usados DURANTE o desenvolvimento de uma versao.
NAO sao necessarios para a atualizacao em si.

## Estrutura

- v1.1.0 -- **arquivado** em `banco-dados/obsoleto/planejamento-v1.1.0-2026-04-10/` (higiene documental; conteudo historico preservado)
- v1.2.0/ -- Planejamento da versao 1.2.0
  - PLANO.md -- Sistema de tasks para rastreamento de demandas

## Como criar planejamento para nova versao

1. Crie pasta {X.Y.Z}/
2. Documente specs, teste, revise
3. Quando pronto, rode scripts\gerar-atualizacao.ps1 para publicar
